

## share-your-places project

### This is a full MERN stack website that let users do multiple functionalities :-

1) SignUp 
2) SignIn
3) SignOut
4) upload there favorite places so other users can see it
5) Edit places info
6) delete places
7) Google  maps & geo APIs integrated
